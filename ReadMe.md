### 2.How to use it?
- First, you need an update package.
- Command:
```basic
Options:
  -help  Show this help message.
  -version  Show version information.
  -gamepath Set the Minecraft game version directory to update.
UpdatepackType:
  -old  Updatepack in Python Script format.(Old format)
  -new  Updatepack in Clang format.(New format)
```
for example:
```basic
update -new test.updatepack
```
